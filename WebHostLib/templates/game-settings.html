Game Settings

This page is used to configure your game settings. You have three presets you can control, which you can access using the dropdown menu below. These settings will be usable when generating a single player game, or you can export them to a .yaml file and use them in a multiworld.
Choose a preset and optionally assign it a nickname, which will be used as the file's description if you download it.
Choose a name you want to represent you in-game. This will appear when you send items to other people in multiworld games.
Glitches Required - Allows the generator to place required items in locations which require knowledge of glitches.
